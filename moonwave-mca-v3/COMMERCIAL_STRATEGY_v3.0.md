# Moonwave MCA v3.0 상용화 전략

## Executive Summary

**현재 상태**: 기술적 기반은 우수하나 핵심 기능 미완성 (완성도 45%)
**목표**: 유료 구독을 유도할 수 있는 차별화된 프리미엄 투자 도구
**전략**: "개인 투자자의 AI 투자 비서" 포지셔닝

### 1.2 Critical Gaps (치명적 누락)

```
🔴 BLOCKER: 상용화 불가능
├── MCA 매매 테이블 - 플레이스홀더만 존재
├── MCA 차트 시각화 - 플레이스홀더만 존재
├── 포트폴리오 파라미터 수정 - 버튼만 존재
└── 주문/체결 체크박스 UI - DB만 존재

🟡 HIGH: 프로덕션 출시 불가
├── 데이터 Import 기능 미구현
├── 체결일 기록 UI 없음
├── 목표가 시뮬레이션 없음
└── 예산 자동 최적화 없음
```

### 1.3 경쟁 환경 분석

| 경쟁자 | 강점 | 약점 | MCA 차별화 포인트 |
|--------|------|------|------------------|
| 토스증권 | UX, 접근성 | 전략 도구 없음 | 분할매수 전략 특화 |
| 키움 영웅문 | 기능 풍부 | 복잡한 UI | 직관적 전략 관리 |
| 삼성POP | 안정성 | 개인화 부족 | 맞춤형 매수 계획 |
| 미래에셋 M-Stock | 리서치 | 전략 자동화 없음 | 자동 최적화 |
| 해외: TradingView | 차트 분석 | 실행 추적 없음 | 주문-체결 추적 |

**MCA 유니크 밸류**: 분할매수 전략을 "계획→주문→체결→평가"까지 End-to-End로 관리하는 유일한 도구

## Part 2: 상용화 비전

### 2.1 Product Vision Statement

```
"Moonwave MCA는 개인 투자자가 감정에 휩쓸리지 않고
수학적으로 검증된 분할매수 전략을 실행할 수 있게 돕는
개인 투자 비서입니다."
```

### 2.2 Target Persona

**Primary: "신중한 개미" (전체 유저의 60%)**
- 연령: 30-50대
- 투자 경험: 1-5년
- 투자 금액: 1천만원 ~ 1억원
- 특징: 하락장에서 매수 타이밍을 놓치는 경험 다수
- Pain Point: "언제, 얼마나 사야 하는지 모르겠다"
- Goal: 체계적인 분할매수로 평단가 관리

**Secondary: "전략적 투자자" (전체 유저의 30%)**
- 연령: 25-40대
- 투자 경험: 5년+
- 투자 금액: 5천만원+
- 특징: 이미 자신만의 전략이 있음
- Pain Point: "전략을 체계적으로 기록/추적하기 어렵다"
- Goal: 다양한 종목의 분할매수 전략 통합 관리

**Tertiary: "학습하는 초보" (전체 유저의 10%)**
- 연령: 20-30대
- 투자 경험: 1년 미만
- 투자 금액: 100만원 ~ 1천만원
- 특징: 투자 공부 중, 도구 활용에 관심
- Pain Point: "분할매수가 좋다는데 어떻게 하는지 모르겠다"
- Goal: 분할매수 전략 학습 및 실습
